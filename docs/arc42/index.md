# DevDiary — Architecture Documentation

DevDiary is a personal developer changelog and diary app. Developers log daily work entries tagged by project, attach
mood indicators, links, and free-text notes, then search and filter those entries by content, date range, project, or
tag. The app also serves as a live demonstration of an AI-assisted development workflow using GitHub Copilot and Claude
Code.
